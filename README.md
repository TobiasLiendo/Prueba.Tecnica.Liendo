Aplicación desarrollada con **Angular 19** y **PrimeNG 19**, que consume datos desde JSONPlaceholder y los muestra en una tabla con paginación, búsqueda por nombre o ciudad, 
y estilos personalizados.

**Instrucciones para ejecutar**

En el visual studio code, crear una nueva terminal, pararse sobre la carpeta del proyecto y ejecutar el comando "ng serve -o". Esto hara que muestre la pagina directamente, 
disponible en http://localhost:4200
